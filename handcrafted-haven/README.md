# Handcrafted Haven

## Project Overview
Handcrafted Haven is a full-stack marketplace web application that allows artisans to create seller profiles, list handcrafted items for sale, and connect with users who want to browse, filter, and review handmade products.

## Functional Features
- Seller profiles (authenticated sellers)
- Product listings with images, descriptions, and pricing
- Browse and filter products by category and price
- Reviews and ratings system

## Design Requirements
- Responsive design for mobile, tablet, and desktop
- Accessibility compliant (WCAG 2.1 AA)
- SEO-friendly structure and metadata
- Consistent branding and navigation

## Technology Stack
- Front-End: Next.js, HTML, CSS, JavaScript
- Back-End: Node.js + Database
- Project Management: GitHub Projects (Kanban)
- Deployment: Vercel
